JAnki is a simple library to use Anki Overdrive with Java.

More information can be found in the JavaDoc of package de.pdbm.janki.ble.

TinyB Library:
You have to use the tinyb.jar suitable for you shared library (*.so).
Copy it to the lib folder and install it in your local maven repository.
To install tinyb.jar from the lib folder to the local maven repository use:
mvn install:install-file@install-external
